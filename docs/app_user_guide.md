# DeepDCR Workstation user guide

The DeepDCR Workstation is a research-use secure web application for case-level review of model outputs.

## Main modules

1. Login and access logging.
2. De-identified CT-DCG upload.
3. Two-stage segmentation.
4. CT and segmentation overlay.
5. Interactive 3D segmentation.
6. Bone-window thickness map.
7. Structured feature table.
8. Surgical difficulty prediction.
9. Modular report export.

## Report export

The report export module generates a structured case-level report containing:

- CT-DCG axial overlay.
- Automated segmentation-derived measurements.
- Bone-window thickness map.
- Three-dimensional anatomical views.
- Surgical difficulty probability.
- Exportable PDF, PNG and ZIP report package.

## Disclaimer

The workstation is for institutional research review only and is not intended for autonomous clinical decision-making.
