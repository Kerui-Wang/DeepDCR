# Installation

This repository provides research-use source code for DeepDCR.

## Recommended environment

- Python 3.10
- CUDA-compatible GPU for nnU-Net inference
- PyTorch
- nnU-Net v2
- scikit-learn
- SimpleITK
- NumPy, SciPy, pandas and scikit-image

## Basic installation

```bash
git clone https://github.com/Kerui-Wang/DeepDCR.git
cd DeepDCR
pip install -r requirements.txt

Model weights are not included in the public repository and must be requested separately for academic research.
